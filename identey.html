02 — Logo & Mark / الشعار
The BOOKLY Mark
علامة بوكلي

The BOOKLY logo combines a stylized calendar icon with the letter "B" — representing booking and beauty in one elegant mark. The scissor motif within the icon subtly references the salon world.

The Wordmark is set in IBM Plex Sans Arabic Bold with generous letter-spacing to give it a modern, app-like presence.

يجمع شعار بوكلي بين أيقونة تقويم مُصمَّمة وحرف "B" — ليمثّل الحجز والجمال في علامة أنيقة واحدة. وتُشير أداة المقص داخل الأيقونة بشكل خفي إلى عالم الصالونات.

الاسم مكتوب بخط IBM Plex Sans Arabic Bold مع مسافات بين الحروف لمنحه حضوراً عصرياً.

05 — Logo Usage / استخدام الشعار
Rules & Guidelines
القواعد والإرشادات
✓
Use on approved backgrounds / استخدمه على خلفيات معتمدة
Deep purple, cream, white, or the gold accent background only. — البنفسجي العميق، الكريمي، الأبيض، أو الذهبي فقط.
✓
Keep clear space / احتفظ بمساحة واضحة
Minimum clear space = height of the letter "B" on all sides. — أدنى مسافة واضحة = ارتفاع حرف "B" من جميع الجهات.
✓
Use IBM Plex Sans Arabic consistently / استخدم الخط باتساق
All brand touchpoints must use IBM Plex Sans Arabic. — يجب أن تستخدم جميع نقاط اتصال العلامة هذا الخط.
✗
Don't distort / لا تشوه
Never stretch, compress, or rotate the logo. — لا تمدّ أو تضغط أو تدوّر الشعار.
✗
Don't use on busy backgrounds / لا تستخدم على خلفيات مشغولة
Never place over photographs or complex patterns. — لا تضعه فوق الصور أو الأنماط المعقدة.
✗
Don't change colors / لا تغير الألوان
Only use approved color combinations. — استخدم مجموعات الألوان المعتمدة فقط.
